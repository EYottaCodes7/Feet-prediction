## project name
Feet-prediction

## Overview

Kids’ feet grow fast—up to 1-2 sizes a year—making online shoe shopping a guessing game, with over 30% of returns due to fit issues.

To help with this, I made a program where parents input their child’s age and foot length (cm) and get instant US, UK, and EU sizes.

I used Adidas's Kids Shoes Size Chart (https://www.adidas.com/us/help/size_charts/kids-shoes) as a reference.

## how it works

input age of a child and their foot length (cm) output shoe size (in US, UK, and EU).
